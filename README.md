# FINANÇAS ON — Site Oficial

Site institucional do **FINANÇAS ON**, desenvolvido para GitHub Pages.

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub chamado `financas-on-site` (ou qualquer nome)
2. Faça upload do arquivo `index.html` para a raiz do repositório
3. Vá em **Settings → Pages**
4. Em **Source**, selecione **Deploy from a branch**
5. Selecione a branch `main` e pasta `/ (root)`
6. Clique em **Save**
7. Aguarde ~2 minutos — o site estará disponível em:
   `https://SEU_USUARIO.github.io/financas-on-site`

## Domínio personalizado (opcional)

Para usar um domínio próprio (ex: `financason.com.br`):
1. Em **Settings → Pages → Custom domain**, insira seu domínio
2. Configure o DNS do seu domínio com um CNAME apontando para `SEU_USUARIO.github.io`

## Atualizar o link de download

No `index.html`, procure por `href="#"` nos botões de download (`.dl-btn`) e substitua pelo link real do instalador.
