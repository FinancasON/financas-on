Como funciona

Organização financeira
sem complicação

O FINANÇAS ON foi pensado para abrir, registrar, acompanhar e sincronizar seus dados de forma simples.

1

Entre com sua conta Google

O acesso é feito de forma simples, usando sua conta Google. Nada de senha separada dentro do app.

2

Registre sua rotina

Cadastre despesas, notas e eventos de agenda no mesmo ambiente, sem trocar de ferramenta.

3

Acompanhe os gráficos

Veja o mês atual, comparativos, evolução dos últimos meses e alertas de vencimento.

4

Sincronize no servidor do app

Seus dados ficam organizados no servidor, com estrutura pensada para privacidade e segurança.

Perguntas frequentes

Dúvidas rápidas
sobre o FINANÇAS ON

Respostas diretas para quem quer baixar, instalar ou entender o funcionamento do app.

Onde faço o download?

Use a seção Downloads da página.

Preciso criar senha?

Não. O acesso do app foi planejado para usar conta Google, evitando senha separada dentro do sistema.

Onde meus dados ficam salvos?

O app trabalha com sincronização pelo servidor próprio, mantendo os dados vinculados à conta utilizada no aplicativo.

Tem versão para Android?

A versão Android está prevista como evolução do projeto. O foco atual do site é apresentar e disponibilizar a versão Windows.

Como faço para atualizar o app?

Assim que uma nova atualização estiver disponivel ela será postada e feita de forma automática pelo aplicativo.
